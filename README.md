# Script de Inspeção mobile (Python + ADB)

Este projeto é uma ferramenta de automação simples para extrair métricas de hardware de dispositivos Android via ADB. 
Ideal para agilizar coletas de dados.

### Funcionalidades
- Identificação automática de modelo e versão de SO.
- Captura de nível de bateria em tempo real.
- Geração de log automático com timestamp para auditoria de testes.

### Como usar
1. Tenha o Python e o ADB instalados.
2. Conecte um dispositivo Android com a Depuração USB ativa.
3. Execute: `python main.py`
